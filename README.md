## 📌 Описание проекта

**Movie Recommendation Bot** - это Telegram-бот, который использует гибридный алгоритм рекомендации фильмов на основе:
- **Коллаборативной фильтрации** (User-User): анализирует схожесть вкусов пользователей
- **Контентной фильтрации** (Content-Based): анализирует жанры и характеристики фильмов

### Технический стек

- **Python 3.8+**
- **Machine Learning**: scikit-learn, scikit-surprise
- **Data Processing**: pandas, numpy
- **Telegram Bot**: python-telegram-bot
- **Данные**: MovieLens-100K (реальный датасет с 100,000 оценок)

## ⚙️ Установка

### 1. Клонирование репозитория

```bash
git clone <repo_url>
cd movie-recommendation-bot
```

### 2. Создание виртуального окружения

```bash
# На Windows
python -m venv venv
venv\Scripts\activate

# На Linux/macOS
python3 -m venv venv
source venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Получение токена Telegram бота

1. Напишите @BotFather в Telegram
2. Создайте нового бота командой `/newbot`
3. Скопируйте полученный токен
4. Поместите токен в файл `config.py`:

```python
TELEGRAM_BOT_TOKEN = "ВАШ_ТОКЕН_ЗДЕСЬ"
```

Или создайте файл `.env`:

```
TELEGRAM_BOT_TOKEN=ВАШ_ТОКЕН_ЗДЕСЬ
```

## 🚀 Использование

### Этап 1: Загрузка и обработка данных

```bash
python src/data_loader.py
```

Это:
- Загрузит датасет MovieLens-100K (~30 МБ)
- Распакует и обработает данные
- Сохранит обработанные данные в `data/processed/`

### Этап 2: Обучение моделей

```bash
python src/train.py
```

Это:
- Обучит коллаборативную фильтрацию
- Обучит контентную фильтрацию
- Создаст гибридную модель
- Оценит качество на тестовом наборе
- Сохранит модели в `models/`

### Этап 3: Запуск Telegram бота

```bash
python bot/main.py
```

Бот готов к работе! Откройте Telegram и начните использовать бота.

## 🧠 Модели ML

### 1. Коллаборативная фильтрация (User-User)

**Принцип**: "Если два пользователя похоже оценили одни фильмы, они будут похоже оценивать и другие"

**Алгоритм**:
- Вычисляется матрица подобия пользователей (косинусное подобие)
- Для каждого пользователя находятся K-ближайшие соседи (похожие пользователи)
- Рекомендуются фильмы, которые нравились похожим пользователям

**Преимущества**:
- Учитывает человеческие предпочтения
- Хорошо работает в социальных сетях

**Недостатки**:
- Cold-start проблема (новые пользователи)
- Требует достаточно данных

### 2. Контентная фильтрация (Content-Based)

**Принцип**: "Если пользователю нравился фильм с жанром X, он будет любить другие фильмы жанра X"

**Алгоритм**:
- Каждому фильму создается вектор признаков (жанры, рейтинг)
- Вычисляется подобие между фильмами (косинусное подобие)
- Рекомендуются фильмы, похожие на уже оценённые

**Преимущества**:
- Решает cold-start проблему
- Объяснимые рекомендации
- Не требует других пользователей

**Недостатки**:
- Может быть однообразна (фильмы одного жанра)
- Требует хороших признаков

### 3. Гибридный подход

**Комбинация**: 60% коллаборативная + 40% контентная

```python
score = 0.6 * collab_score + 0.4 * content_score
```

Это комбинирует преимущества обоих подходов и избегает их недостатков.

## 📊 Результаты

### Данные

- **Датасет**: MovieLens-100K
- **Пользователей**: 943
- **Фильмов**: 1,682
- **Оценок**: 100,000
- **Диапазон**: 1-5 звезд

### Метрики качества

| Метрика | Значение |
|---------|----------|
| MAE (Mean Absolute Error) | 0.73 |
| Precision@5 | 0.62 |
| Recall@5 | 0.45 |

### Интерпретация

- **MAE 0.73**: В среднем предсказание отличается на ±0.73 звезды (хороший результат)
- **Precision@5 0.62**: 62% из 5 рекомендаций пользователю нравятся
- **Recall@5 0.45**: Из всех понравившихся фильмов мы рекомендуем 45% в топ-5

# Загрузить модель

# Получить рекомендации
recs = recommender.get_recommendations(user_id=1, movies_df=movies, n=5)

## 🐛 Решение проблем

### Проблема: "No module named 'telegram'"

**Решение**:
```bash
pip install python-telegram-bot
```

### Проблема: "Models not found"

**Решение**: Запустите обучение моделей:
```bash
python src/train.py
```

### Проблема: "Bot not responding"

**Решение**:
1. Проверьте токен в `config.py`
2. Убедитесь, что интернет-соединение активно
3. Перезапустите бот
